# Trial Tracker
track clinical trials for any medical condition.
inspired by https://hairloss.pages.dev/

## Project Features
- clinical status (phase 1, phase 2, etc) of ALL drugs for a condition.
- extra information (company, phase, type, expected release, status, mechanism, summary, latest)
- seperate view showing existing treatments with more info like efficacy, side effects, long term effects, etc
- updates with a recurring batch job (daily? weekly?)
- pulls data from clinicaltrials.gov
- explore other data sources such as press releases, reddit, etc
- all 10,000+ medical conditions are tracked
- search bar
- email notifications for updates on specific conditions
- website hosted and publically available
